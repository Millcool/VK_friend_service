
Примеры вызова API:

Запрос GET на главную страницу:
URL: http://localhost:8000/
Ответ: Отобразится страница приветствия.

Запрос POST на страницу входа пользователя:
URL: http://localhost:8000/users/log_in/
Параметры запроса: username - имя пользователя
Ответ: Если пользователь с указанным именем существует, будет отображена страница пользователя с его данными. В противном случае будет отображена страница регистрации пользователя.

Запрос POST на страницу регистрации пользователя:
URL: http://localhost:8000/users/register/
Параметры запроса: username - имя пользователя
Ответ: Если пользователь с указанным именем уже существует, будет отображена страница с сообщением об ошибке. В противном случае будет отображена страница входа пользователя.


Запрос GET на страницу пользователя:
URL: http://localhost:8000/user/{user_id}/user_page/
Ответ: Отобразится страница с информацией о пользователе и его друзьях.

Запрос GET на страницу списка друзей:
URL: http://localhost:8000/{user_id}/friend_list
Ответ: Отобразится страница со списком друзей пользователя.


Запрос GET на страницу запросов в друзья:
URL: http://localhost:8000/{user_id}/friend_requests/
Ответ: Отобразится страница со списком запросов в друзья для пользователя.


Запрос POST на принятие/отклонение запроса в друзья:
URL: http://localhost:8000/{user_id}/{friend_id}/{accepted}/add_friend/
Параметры запроса: accepted - значение 1 (принятие) или 0 (отклонение)
Ответ: Будет выполнено соответствующее действие (принятие или отклонение запроса) и отображена страница пользователя.

Запрос POST отзыв заявки в друзья:
URL: http://localhost:8000/{user_id}/{friend_id}/withdraw/
Ответ: Будет выполен отзыва и удаление заявки из базы данных и отображена страница пользователя.

Запрос POST на удаление друга:
URL: http://localhost:8000/{user_id}/{friend_id}/remove_friend/
Ответ: Отобразится страница пользователя после удаления друга, и соответствующая запись будет удалена из базы данных.


Примеры запуска сервиса:

    Запуск сервера разработки:

    В командной строке перейдите в корневую папку проекта.
    Выполните команду python manage.py runserver.
    Сервер разработки будет запущен и доступен по адресу http://localhost:8000/.

В приведенном коде присутствуют операции с базой данных SQLite, поэтому
 перед запуском сервиса убедитесь, что на вашей системе настроено подключение
 к базе данных SQLite и выполнены необходимые миграции.

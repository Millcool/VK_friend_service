Краткая документация:

Описание сервиса: Данный сервис предоставляет функциональность для управления друзьями пользователей.

Установка и настройка:
Установите все зависимости, указанные в файле requirements.txt.

Создайте базу данных и настройте подключение к ней в файле settings.py.

Выполните миграции базы данных с помощью команды python manage.py migrate.

Запустите сервер разработки с помощью команды python manage.py runserver.

Сервис будет доступен по адресу http://localhost:8000/.
